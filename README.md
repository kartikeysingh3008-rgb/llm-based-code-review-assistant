# llm-based-code-review-assistant
LLM Based Code Reviewer that uses Python's AST to understand the code with a rule based engine, followed by a review generated from Groq (llama-instant).

*Note: In order to use the LLM, you need to use your own API key and store it in the .env folder for the code to work

Check out the live website on:
https://code-review-assistant-pixi.onrender.com
